# Yahtzee Coach: Hold Strategy Trainer

A focused Streamlit web app for practicing Yahtzee Roll 1 and Roll 2 dice-hold decisions.

## Scope

This app intentionally does **not** include Roll 3 scoring/category-choice decisions. It focuses on one skill: choosing which dice to hold and understanding why.

## Files

- `app.py` — Streamlit interface
- `yahtzee_engine.py` — exported engine/report code from the July 8 clean notebook
- `requirements.txt` — dependencies for Streamlit Community Cloud

## Deploy

Use Streamlit Community Cloud. Main file path: `app.py`.
